# Jack Tang — Personal Webpage

This repository contains a single-page personal website for Jack Tang with a modern glassmorphism design and a live clock.

Features
- Live clock updating every second (HH:MM:SS)
- Full date display (e.g., Tuesday, February 24, 2026)
- Name shown with a purple gradient shimmer effect
- Glassmorphism card with frosted-glass backdrop blur
- Animated deep purple/indigo gradient background
- Three floating blurred orbs for depth
- Smooth staggered fade-up entrance animations
- Hover lift effect on the card
- Responsive layout for mobile and desktop
- Google Fonts: Outfit & Inter

Files
- `index.html` — main page (open in browser)
- `styles.css` — styling: background, glass card, animations
- `script.js` — live clock and entrance timing

Usage
1. Open `index.html` in your browser (double-click or serve with a static server).

Optional local server (recommended for development):

```bash
# using Python 3
python -m http.server 8000
# then open http://localhost:8000
```

Notes
- Fonts are loaded via Google Fonts; an internet connection is required for fonts unless you self-host them.
- The design uses CSS backdrop-filter — some older browsers may not support the full frosted-glass effect.

If you want changes (colors, text, add links or socials), tell me which parts to update.
